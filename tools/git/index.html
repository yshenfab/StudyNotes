SSH

生成和添加key

ssh-keygen -t rsa -b 4096 -C "xxxxxx@qq.com" 复制~/.ssh/idrsa.pub的内容里（也就是key） 进入github settings，添加到New SSH key

验证

ssh -T git@github.com 如果看到You've successfully authenticated, but GitHub does not provide shell access表明成功

GPG

生成GPG Key

gpg –gen-key 要求输入Real Name: FirstName LastName，Email: xxxxxxx@qq.com以及密码 生成pub公钥，sub私钥 pub后面的一长串就是GPG Key ID，例如为9639D**92E4A，复制它，然后 gpg –armor –export 9639D**92E4A 会生成 –—BEGIN PGP PUBLIC KEY BLOCK–—

–—END PGP PUBLIC KEY BLOCK–— 进入github settings，添加到New GPG key，复制上方所有内容，粘贴后add GPG key 再配置本地git： git config –global user.signingkey 9639D**92E4A git config –global commit.gpgsign true

basic usage

git init git add / git add . git commit -m "message" git remote add origin https://github.com/yshenfab/repo_name.git git push origin master
