# My Study Notes

> ABC: Always Be Curious.

A collection of my study notes.

## Install

```sh
pip install mkdocs
pip install python-markdown-math
```

## Usage

```sh
mkdocs serve     # http://127.0.0.1:8000 view locally
mkdocs gh-deploy # deploy github pages
```
